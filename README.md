# Data Structures & Algorithms Visualizer

An interactive web-based tool that visually demonstrates how common sorting algorithms work.  
This project helps users understand algorithm behavior through animated step-by-step execution.

## Live Demo

Try the project here:

https://mohit-devbuild.github.io/BasicProject1/

## Features

- Interactive visualization of sorting algorithms
- Step-by-step execution of algorithm operations
- Demonstrates algorithm behavior and sorting mechanics
- Educational tool for understanding data structures and algorithms

## Algorithms Implemented

The visualizer currently supports the following algorithms:

- Bubble Sort
- Insertion Sort
- Selection Sort
- Merge Sort
- Quick Sort
- Bogo Sort

Each algorithm is animated to show how elements are compared, swapped, and arranged during the sorting process.
